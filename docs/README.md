# RWPM - Railway Work Progress Monitoring System

## Overview
RWPM is a digital transformation platform for Indian Railways engineering reports. It streamlines the creation, approval, and management of daily progress reports, ensuring standard compliance and data-driven insights.

## Key Features
- **Multi-Step Structured Input:** Reduces complexity and input errors.
- **Automated Weather Integration:** Syncs local site weather for environmental logging.
- **Official Railway Format PDFs:** One-click generation of divisional reports.
- **Google Sheets Backend:** Cost-effective, auditable, and familiar data storage.
- **Admin/JE Hierarchy:** Role-based access mirroring on-site reporting structures.

## Tech Stack (Proposed)
- **Frontend:** React + Vite (for speed and responsiveness)
- **State Management:** Logic-heavy multi-step forms
- **Backend/API:** Google Apps Script / Node.js
- **Database:** Google Sheets API
- **PDF Generation:** jsPDF / React-PDF
- **Styling:** Vanilla CSS with Premium UI/UX
